# Drug Conflict Intelligence RAG

AI-powered Drug-Drug Interaction Detection using PubMedBERT, FAISS, and Retrieval-Augmented Generation (RAG).

## Features

- Multi-drug interaction analysis
- PubMedBERT interaction prediction
- FAISS semantic retrieval
- Biomedical evidence retrieval
- Risk scoring
- Explainable AI
- Polypharmacy support
